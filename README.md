# LaUkl2B
Program ini berfungsi untuk menentukan apakah suatu bilangan yang dimasukkan oleh pengguna merupakan bilangan ganjil atau genap. Pengguna memasukkan sebuah angka, lalu program memeriksa sisa hasil bagi angka tersebut dengan 2 menggunakan operator modulus (%). Jika hasilnya 0 maka bilangan genap, jika tidak maka bilangan ganjil.
